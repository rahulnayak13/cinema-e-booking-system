# Cinema E-Booking System 

CSCI 4050/6050 – Software Engineering  
Team 17  

## Tech Stack
- Frontend: React (Vite)
- Backend: Spring Boot
- Database: MySQL

## How to Run

### Backend
cd backend
./mvnw spring-boot:run

### Frontend
cd frontend
npm install
npm run dev

Frontend: http://localhost:5173  
Backend API: http://localhost:8080/api/movies
